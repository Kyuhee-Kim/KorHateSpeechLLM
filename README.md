## 🧠 Overview

Toxic language detection has become an essential task for maintaining healthy online communities.  
This research explores computational approaches for classifying toxic language in Korean using modern NLP models.

The thesis investigates:

- Multi-label toxic language classification
- Korean-specific linguistic challenges
- Transformer-based models
- Evaluation across multiple toxicity categories
